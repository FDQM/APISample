# APISample
AppSample using https://www.thecocktaildb.com/api.php API. The App retrieves a list of drinks depending of the categorie the user chooses.

Categories:
* Ordinary Drink
* Cocktail
* Milk / Float / Shake
* Other/Unknown
* Cocoa
* Shot
* Coffee / Tea
* Homemade Liqueur
* Punch / Party Drink
* Beer
* Soft Drink / Soda
